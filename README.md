# BuildingBloCS Dec 2025 - SaveTheWorld

This website is a very simple website that provides easy tips for anyone wanting to save the world, with the convenient click of a button. If more tips are needed, just return to the home page and click the button again for another tip. Good luck!

## Functionalities

A moving button that will provide tips when (if) clicked. 

A backend server made using Flask that provides a random tip out of a text file when requested. 

## Setup 
Prerequisites: Have Python 3.12 installed. 

1. If not installed, install Flask 3.1.

With pip: `pip install flask`

2. In a terminal in the project folder, run `python app.py`

3. Browse to http://127.0.0.1:5000 to access the website.

## AI Usage
ChatGPT was used to generate the path for the button to move in CSS.
