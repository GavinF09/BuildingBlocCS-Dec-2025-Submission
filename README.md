# SaveTheWorld

A little website with tips to save the world :). But that would be too simple, so we had a bit of fun with the button

## Functionalities

A moving button that will provide tips when (if) clicked. 

A backend server made using Flask that provides a random tip out of a text file when requested. 

## Setup 
Prerequisites: Have Python 3.12 installed. 

1. If not installed, install Flask 3.1.

With pip: `pip install flask`

2. In a terminal in the project folder, run `python app.py`

3. Browse to http://127.0.0.1:5000 to access the website.
